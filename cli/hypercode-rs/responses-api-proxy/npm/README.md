# @openai/hypercode-responses-api-proxy

<p align="center"><code>npm i -g @openai/hypercode-responses-api-proxy</code> to install <code>hypercode-responses-api-proxy</code></p>

This package distributes the prebuilt [Hypercode Responses API proxy binary](https://github.com/openai/hypercode/tree/main/hypercode-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/hypercode-responses-api-proxy.js --help
```

Refer to [`hypercode-rs/responses-api-proxy/README.md`](https://github.com/openai/hypercode/blob/main/hypercode-rs/responses-api-proxy/README.md) for detailed documentation.
